# Security Policy

## Supported Versions

This project uses semver and the current stable version is being supported

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.x.x   | :x:                |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Please use github issues to report any vulnerability
