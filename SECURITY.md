# Security Policy

## Supported Versions

Only the current major version is supported

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :x:                |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Please use github issues to report any vulnerability
