# Security Policy

## Supported Versions

Only the current minor version is supported

| Version | Supported          |
|---------|--------------------|
| 1.5.x   | :white_check_mark: |
| 1.4.x   | :x:                |
| 1.3.x   | :x:                |
| 1.2.x   | :x:                |
| 1.1.x   | :x:                |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Please use github issues to report any vulnerability
